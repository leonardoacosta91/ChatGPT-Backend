# ChatGPT Backend

Prueba sencilla de chateo con ChatGPT usando Flask

### Pre-requisitos 📋
Docker

## Despliegue 📦
Renombrar archivo .env.example a .env y agregar APIKey de OpenAI
Ejecutar:
```
docker-compose build
docker-compose up
```

## Puerto de la app ⚓
El proyecto se ejecuta en http://localhost:8000
